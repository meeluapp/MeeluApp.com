Frequently Asked Questions

How does Meelu keep my exchange data secure?

Meelu uses Apple's Keychain technology to securely store your exchange API keys. Your private keys never leave your device, and all sensitive data remains encrypted. We don't store your data on our servers, ensuring your crypto assets remain under your control.

How do I connect my exchange accounts?

To connect an exchange, navigate to the Settings menu, select "Add Exchange," and choose your exchange from the list. You'll need to generate read-only API keys from your exchange account and enter them in Meelu. We recommend using read-only API keys without withdrawal permissions for maximum security.

Why do I need to use API keys?

API keys allow Meelu to securely connect to your exchange accounts without storing your actual exchange login credentials. This is much safer than providing your username and password, as you can restrict what the API key has access to (we only need the ability to view balances, not make trades or withdrawals).

Which exchanges are currently supported?

Meelu currently supports Binance, Coinbase, Kraken, Kucoin, and other major exchanges. We're constantly working to add more exchanges based on user demand.

How do I customize what counts as "cash" in my portfolio?

Go to Settings > Display Preferences > Asset Classification. Here you can define which assets should be counted as "cash" in your portfolio view. By default, stablecoins like USDT, USDC, and DAI are classified as cash.

How often are balances updated?

Meelu refreshes your balances automatically when you open the app. You can also manually refresh by pulling down on the main portfolio screen. In the settings, you can adjust how frequently the app checks for updates while running in the background.

How do I set up low-limit filters?

Go to Settings > Filters > Low Value Assets. Here you can set a minimum USD value threshold. Assets below this value can be filtered out of your main view to reduce clutter. You can toggle this filter on/off from the main portfolio screen.

Troubleshooting

Connection Issues

If you're having trouble connecting to an exchange:

  • Verify your API keys are entered correctly
  • Ensure your API keys have read permission for account balances
  • Check if your exchange is experiencing downtime
  • Try removing and re-adding the exchange

Balance Discrepancies

If you notice differences between Meelu and your exchange:

  • Ensure all subaccounts are properly connected
  • Try a manual refresh by pulling down on the portfolio screen
  • Check if you have any open orders that might affect available balance
  • For small differences, check if your exchange has any hidden fees or dust amounts
